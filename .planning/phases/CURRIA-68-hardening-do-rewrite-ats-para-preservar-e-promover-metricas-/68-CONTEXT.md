# Phase 68 Context

## Goal

Harden ATS enhancement rewriting so strong quantified impact from the original resume is preserved or promoted instead of being flattened into generic ATS-friendly bullets.

## In Scope

- Rewrite prompt guardrails for quantified impact
- Lightweight premium-bullet detection for high-value metrics
- Validation gate for editorial metric regression before persistence
- Regression tests for percent-based and volume-based impact bullets

## Out of Scope

- ATS Readiness contract changes
- Floor 89, cap 95, or exact-vs-estimated-range changes
- Heavy semantic NLP or outcome prediction
