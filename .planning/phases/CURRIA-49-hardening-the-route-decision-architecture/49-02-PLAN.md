| Boundary | Description |
|----------|-------------|
| request parsing -> route context | Untrusted input becomes typed route context and must not accumulate hidden business semantics |
| decision layer -> response layer | Normalized route outcomes cross into public payload mapping and must not leak locked or billable artifacts |
| policy gate -> execution orchestration | Precedence-sensitive blocking must happen before durable job creation or artifact retrieval |
