# Quick Task 260424-d2p - Renomear endpoints de auth/dashboard para a nova navegação canônica

## Goal

Trocar os endpoints legados do fluxo autenticado para a nova estrutura canônica:

- `/dashboard/resumes/new` e `/dashboard/resume/new` -> `/profile-setup`
- `/dashboard` -> `/chat`
- página de histórico -> `/dashboard/resumes-history`
- redirects, links internos e `redirect_to` precisam respeitar essas rotas

- `src/app/(auth)/dashboard/resumes-history/page.tsx`

**Action**
- Criar as rotas canônicas novas.

**Verify**
